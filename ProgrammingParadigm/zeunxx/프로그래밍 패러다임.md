# 프로그래밍 패러다임

## 프로그래밍 패러다임이란?

프로그래머에게 프로그래밍의 관점을 갖게 해주는 역할을 하는 개발 방법론



특정 언어는 특정한 패러다임을 지원하기도 한다.
- jdk 1.8 이전의 자바: 객체 지향 프로그래밍 지원
- 하스켈 : 함수형 프로그래밍 지원


여러 패러다임을 지원하는 언어
- c++
- python


jdk 1.8부터의 자바의 경우 
- 함수형 프로그래밍 패러다임을 지원하기 위해 람다식, 생성자 레퍼런스, 메서드 레퍼런스 도입
- 선언형 프로그래밍 패러다임을 지원하기 위해 스트림(stream) 같은 표준 API 도입


## 프로그래밍 패러다임의 종류

1. 선언형
    - 함수형

2. 명령형 
    - 객체 지향형
    - 절차 지향형
