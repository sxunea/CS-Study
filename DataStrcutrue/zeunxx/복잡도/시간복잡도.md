# 시간 복잡도

## 빅오 표기법

**시간복잡도** : 문제를 해결하는데 걸리는 시간과 입력의 함수 관계

➡️ 어떤한 알고리즘의 로직이 *얼마나 오랜 시간*이 걸리는지 나타내는데 쓰임 = 알고리즘을 수행하는 데 연산들이 몇 번 이루어지는 지



<br>

빅오 표기법이란 입력범위 n을 기준으로 해서 **로직이 몇번 반복되는지** 나타내는 것

➡️ 가징 영향을 많이 끼치는 함수만을 고려
