# 속성

속성 attribute은 릴레이션에서 관리하는 구체적이며 고유한 이름을 갖는 정보
- 데이터 모델링에서 속성은 엔티티의 정보를 나타내고, **더 이상 분리되지 않는 최소의 단위**
    - 속성은 하나의 값만 가져야 하기 때문
    - 그래야 해당 속성에 저장되는 데이터가 의미가 있음
